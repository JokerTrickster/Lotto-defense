# Game Font

Place your game font here so the whole game uses it.

- **File name:** `GameFont.ttf` (or import the font and name the Unity asset "GameFont" in this folder)
- **Suggested style:** Rounded, readable fonts that fit a cute/casual game (e.g. [Nunito](https://fonts.google.com/specimen/Nunito), [Varela Round](https://fonts.google.com/specimen/Varela+Round), [Quicksand](https://fonts.google.com/specimen/Quicksand) from Google Fonts).

If no custom font is present, the game falls back to Unity's built-in font.
